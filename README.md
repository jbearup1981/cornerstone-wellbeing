# Cornerstone Well-Being Program

Working project hub for the Cornerstone Well-Being Program — a Section 125 fixed-indemnity wellbeing program delivered through Nexus Benefit Solutions, with Insight Benefits as TPA and Capitol Group as carrier/captive/reinsurance rails. Powered by Amaze Health.

## What this repo is

The single-page project hub that documents the program's sales / implementation / ongoing service processes, supporting artifacts (eligibility file specs, sample invoices, contract package, payroll system support matrix), and operational handoffs between Nexus, Capitol, Insight, and Amaze.

Hosted via GitHub Pages at: <https://jbearup1981.github.io/cornerstone-wellbeing/>
